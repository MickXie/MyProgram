RSI 計算流程（對應程式碼）
delta = close.diff()
gain = delta.clip(lower=0)
loss = (-delta).clip(lower=0)

步驟	說明
delta	每日漲跌
gain	只保留上漲
loss	只保留下跌
avg_gain = gain.rolling(14).mean()
avg_loss = loss.rolling(14).mean()


為什麼用 rolling mean？

RSI 是「一段期間的平均動能」

rolling window = 14 日是金融分析標準

rs = avg_gain / avg_loss
rsi = 100 - (100 / (1 + rs))

超買 / 超賣判斷依據
RSI 值	判斷
> 70	超買（可能回檔）
< 30	超賣（可能反彈）
30–70	正常區間
